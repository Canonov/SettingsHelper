# SettingsHelper [![RimWorld 1.6](https://img.shields.io/badge/RimWorld-1.6-green.svg?longCache=true&style=plastic)](http://rimworldgame.com/) [![Wiki](https://img.shields.io/badge/documentation-Wiki-4BC51D.svg?style=flat)](https://github.com/Canonov/SettingsHelper/wiki)

A helper library for RimWorld mod developers that simplifies building mod settings UIs. Extends `Listing_Standard` with additional controls and layout utilities.

## Features

- **Form Controls** — labeled text fields, numerical inputs, checkboxes, and sliders (including enum sliders)
- **Layout Helpers** — label lines, horizontal dividers, rect splitting, and multi-column listings
- **Radio Lists** — radio button groups from string arrays or `Dictionary<string, T>`
- **Color Picker** — full HSV color picker dialog with hex input, alpha support, and presets
- **Sub-Settings Windows** — open secondary settings windows from a button with a single call
- **Configurable Spacing** — global `GapValues` for consistent, adjustable spacing
